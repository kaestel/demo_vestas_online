JES Util

Util or short u is the overall JES framework namespace.

Util.getElement

Name
Util.getElement
Shorthand
u.ge
Syntax
Util.getElement(String id [, Mixed target])

Description

Get element (id/class/tag)
Returns elementById if possible
else Returns first element with (partial) matching classname from target
If no matches, return first element with tagname from target

parameters

identifier
String node id, node classname or node tagname
target
Node Optional, any given node could be used as scope

Returns

Returns elementById if possible
else Returns first element with (partial) matching classname from target

Examples

u.getElement("content");

returns Node with id/classname content

Util.Request

Name
Util.Request
Shorthand
u.Request
Syntax
Void = u.Request( Node node String url [, string parameters ] [, String method ] [, String async ] );

Description

Make a server request using XMLHttpRequest or <script> injection in <head>.

Makes callback to node.Response(response) when response is received. Declare this function on node to receive callback.

Parameters

node
Node node to attach request to. Response callback will be made to this node.
url
String url to make request to.
parameters
String Optional - parameter string like var=test&var2=hest.
method
String Optional - Request method
Options
POST
POST parameters
GET
Append parameters to url - DEFAULT
SCRIPT
Script injection in header (JSONP)
async
Boolean Optional - set to true for asynchronous request.

Returns

Void

Examples

var node = u.qs("#content"); node.Response = function(response) { var name = response.name; } u.Request(node, "/json");
