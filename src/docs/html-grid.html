Grid system

Grids

We are working with an overall 4 column grid where .c400, .c300, .c200 and .c100 are the most commonly used column sizes. As you can see below there are other possibilities too to allow for a more flexible design.

Columns are created by using div tags with "cnumber" class names inside a div with the class "c".

Example with two columns of equal size:

<div class="c"> <div class="c200"> Left column - 50% width </div> <div class="c200"> Right column - 50% width </div> </div>

Example with a small left column and larger content area on the right:

<div class="c"> <div class="c100"> Left column - 25% width </div> <div class="c300"> Right column - 75% width </div> </div>
