Tabs

You can create a tabbed interface inside the .body area of a .box using an unordered list with .tabs class and the initializer i:tabs.

Each li in the list needs both a .tab class and another class name which must be unique for this tab list. An example of a unique name below is "descriptive_class_name_1".

Each li also needs both a div.header and div.body. The .header must have a h4 tag which is used to create the actual tab-button. The content is inside the .body div is shown as the tab content.

You can see a demo on the tabs demo page.

<ul class="tabs i:tabs"> <li class="tab descriptive_class_name_1"> <div class="header"> <h4>Tab 1</h4> </div> <div class="body"> <p>Tab 1 content</p> </div> </li> <li class="tab descriptive_class_name_2"> <div class="header"> <h4>Tab 2</h4> </div> <div class="body"> <p>Tab 2 content</p> </div> </li> <li class="tab descriptive_class_name_3"> <div class="header"> <h4>Tab 3</h4> </div> <div class="body"> <p>Tab 3 content</p> </div> </li> </ul>
