i:tabs

Tabs

The i:tabs initializer creates a set of tabs based on a <ul> with predefined classnames providing the structural information the initializer needs to create the tab-set.

The HTML being initialized will be rearranged to create the tab-set and the resulting HTML will be different from the HTML served.

Variations

The i:tabs initializer does not provide any options for variations.
