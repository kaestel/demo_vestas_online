JavaScript initializers

Initializers

Initializers is a way of extending your HTML with JavaScript functionality. The page shell is using initializers for many different purposes, such as handling page-sharing and feedback.

An initializer maps a specific JavaScript function with an HTML node, by adding an i:#name# class to the element.

Some initializers are specialized for the page shell, controlling navigation, content loading and global features such as page-sharing and feedback. In addition to the specialized initializers more generic content related components also exists. You can use these in any content-context you wish.
