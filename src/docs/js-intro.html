JES - JavaScript ExtensionS

The site uses JES framework components to extend the HTML. This means each HTML node can be extended simply by adding a initializer class to the HTML tag. The corresponding extension will automatically be invoked when the page is loaded.

JES provides a set of Utilities, which can aid you in writing custom JavaScript, that interacts with the page without conflicting with the overall site flow. This allows you to write custom modules which can be reused very easily.

JES also provides a set of generic components, which extends HTML nodes with frequently used funcionality. If you write the HTML correctly, the rest is taken care of when the content loads.

Finally a set of specialized functions have been written for this project. We call it the Vestas package.
