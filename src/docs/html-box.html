Content - Box

The default content containers are named boxes - You can see a demo of the examples below on the boxes demo page. For information about creating grids inside boxes, see the Grid system page.

<div class="box"> <div class="header"> <h2>Box title</h2> </div> <div class="body"> Box content </div> </div>

Box header

Box header can contain H2 and/or UL.actions with LI.back. Optional content will be extended as need arises.

<div class="box"> <div class="header"> <ul class="actions"> <li class="back"><a href="#link#">Back</a></li> </ul> <h2>Box title</h2> <div class="notification">2 new issues</div> </div> <div class="body"> Box content </div> </div>

Box body

Box body applies standard design for H3, H4, P, UL.list, LI, <dl>, <dt>, <dd> and IMG.picture.

Variations should be based upon #content classname and only apply to specified scope.
